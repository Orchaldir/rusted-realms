= ADR 0001 - Using AsciiDoc for documentation

*Status:* accepted

*Date:* 2019-07-15

== Considered Options

* Markdown
* AsciiDoc

== Decision Outcome

*Chosen option:* AsciiDoc

* AsciiDoc more powerful than Markdown.
* More familiar with AsciiDoc.
